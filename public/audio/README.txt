Place production game audio files in this folder.

The game loads these automatically at runtime:
- bgm_scifi_loop.ogg
- sfx_laser.wav
- sfx_rocket.wav
- sfx_cannon.wav
- sfx_explosion_small.wav
- sfx_explosion_big.wav
- sfx_shoot.wav
- sfx_hit.wav
- sfx_combo.wav
- sfx_levelup.wav
- sfx_gameover.wav
- sfx_damage.wav
- sfx_ui_select.wav
- sfx_ui_tower_select.wav
- sfx_ui_swap.wav
- sfx_ui_clear.wav
- sfx_countdown.wav
- sfx_whoosh.wav
- sfx_score.wav

If any file is missing or fails to play, the game falls back to procedural synth audio for that sound.
